⚙️ Configurar ClipSync

Cole as credenciais do seu projeto Firebase abaixo. Você só faz isso uma vez.

1
Acesse console.firebase.google.com → Adicionar projeto
2
Dê um nome (ex: clipsync), desative o Google Analytics e clique em Criar projeto
3
No menu lateral: Firestore Database → Criar banco de dados → Modo de teste → Ativar
4
Volte ao início, clique no ícone </>, dê um apelido e clique em Registrar app
5
Copie os valores do bloco firebaseConfig e cole abaixo
Preencha todos os campos.
